Life OS
A personal productivity operating system built entirely in React. Track everything that matters — tasks, habits, goals, focus, and wellbeing — all stored locally in your browser.

What is Life OS?
Life OS is a single-page productivity dashboard that replaces the need for multiple apps. Everything lives in one place, works offline, and stores data directly in your browser with no accounts or servers required.

Features
Dashboard - Daily overview — tasks, habits, focus time, and progress at a glance
Tasks - Create tasks with priority, category, deadline, subtasks, and recurring options
Habits - Track daily habits with streaks, success rate, weekly/monthly/heatmap views
Goals - Set short and long-term goals, link tasks to them, and track progress
Analytics - Charts for task completion, habit consistency, focus minutes, and productivity score
Notes - Capture notes and ideas quickly with tags and search
Focus - Pomodoro-style focus timer with session logging and task linking
Control - Log distractions by type and trigger to identify and break patterns
Calendar - Weekly view with drag-and-drop task scheduling and tomorrow planner
Life Log - Daily mood, energy, and sleep tracker with 7-day trend charts
Content - Content pipeline tracker — from idea to scripted to filmed to posted
Settings - Theme switching, data overview, and full data reset

Themes
Life OS comes with 6 built-in themes that change the entire app color palette instantly.

Emerald — dark green with teal accent
Cobalt — deep navy with electric blue accent
Obsidian — pure black with soft violet accent
Crimson — dark burgundy with rose accent
Amber — warm cream with deep amber accent
Pearl — clean white with bold blue accent

Switch between themes from the sidebar or the Settings page.

Tech Stack

React — UI and state management
Recharts — analytics charts
localStorage — all data persistence, no backend
CSS custom properties — full theme system with zero hardcoded colors

Getting Started
1. Clone the repository
bashgit clone https://github.com/yourusername/life-os.git
cd life-os

2. Install dependencies
npm install

3. Start the development server
npm run dev

4. Open in browser
http://localhost:5173



Data & Privacy

All data is saved in your browser's localStorage
Nothing is sent to any server
No accounts, no login, no tracking
To export or back up your data, open browser DevTools → Application → Local Storage and copy the values
To clear all data, go to Settings → Clear All Data
